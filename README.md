### Описание варианта (ЛР 4.1)
В рамках первой лабораторной работы спроектированы и реализованы **два базовых класса** для ресторанного бизнеса **без использования наследования**:

1. **Базовый класс 1 (Сущность):** `MenuItem`  
   - Обязательный атрибут: `price` (цена)
   - Дополнительные атрибуты: `name` (название), `category` (категория)

2. **Базовый класс 2 (Субъект):** `Customer`  
   - Обязательный атрибут: `table_number` (номер столика)
   - Дополнительные атрибуты: `name` (имя клиента), `guests_count` (количество гостей)

### Реализованный функционал
- **Конструкторы `__init__`** для инициализации всех атрибутов объектов
- **Методы `display_info()`** для вывода информации об объектах в консоль
- **Магические методы `__str__`** для "Vibe Code" (красивого строкового представления объектов через `print()`)
- **Соблюдение стандарта PEP 8**: type hints, docstrings, осмысленные имена переменных
